# command-wrap-notify 

## usage

```
NAME:
   command-wrap-notify - my project template

USAGE:
   command-wrap-notify [global options] command [command options] [arguments...]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --command value  command fullpath.
   --help, -h       show help (default: false)
```
